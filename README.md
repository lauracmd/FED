<details>

  ### Bevindingen
  Lijst met je bevindingen die in de test naar voren kwamen:
  - geen gebruik gemaakt van headings
  - veel gebruik gemaakt van divs
  - geen unieke titels
  - geen unieke buttons, a's en labels -> geen goede alts gebruikt
